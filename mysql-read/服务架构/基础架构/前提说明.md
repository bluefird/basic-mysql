# 1. 主从复制简介    

- 1.1. 基于二进制日志复制的
  1.2. 主库的修改操作会记录二进制日志
  1.3. 从库会请求新的二进制日志并回放,最终达到主从数据同步
  1.4. 主从复制核心功能:
- 辅助备份,处理物理损坏 
- 扩展新型的架构:高可用,高性能,分布式架构等
